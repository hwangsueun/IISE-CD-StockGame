# 동학개미 서바이벌 (ANT SURVIVAL) — 기술 스택 가이드라인

> 스코프: 캡스톤 **중간보고서 기준 풀스코프**. 기술스택은 기존 정의 유지.

## 전체 아키텍처

```
React + Vite  →  Express (Node.js)  →  Docker (PostgreSQL)
```

---

## 프론트엔드

| 항목 | 기술 |
|---|---|
| 프레임워크 | React 19 |
| 빌드 도구 | Vite |
| 언어 | JavaScript (JSX) |
| 스타일 | CSS |

- 게임 기간: **1년 = 240턴** (주말 제외 거래일, 1턴=거래일 하루, 20턴=1개월)
- 투자 자산: **총 131개 — 주식 117 / 채권 4 / 코인 10**
- 초기 자금: **5,000만 원** (개발 중 변경 예정)
- 부채(난이도): **5,000만 / 1억 / 1억 5,000만** 중 선택
- 상태값: 현금 · 총자산 · 부채 · **스트레스(0–100)** · **신뢰도(0–100)**
- 뉴스: 하루 **최대 10건**, 스트레스 구간별 열람 제한
- 주요 화면: 오프닝 · 시작/난이도 · 메인 대시보드 · 투자(주식/채권/코인) · 매수/매도 · 포트폴리오 · 뉴스 · 캘린더 · 부업 · 이벤트 팝업 · 엔딩/리포트

## 백엔드

| 항목 | 기술 |
|---|---|
| 서버 | Express (Node.js, plain JS) |
| 구조 | MVC (routes / controllers / services) |

| 항목 | 기술 |
|---|---|

| LLM | GPT-4o (OpenAI Batch API) |

### 파이프라인 단계

```
pr05d → pr05e → pr05f → pr06a   # 개별주식뉴스 (Batch API)

pr_dci06                         # 종토방 NPC 반응 (Batch API)
```

> 주식 117종목은 FnGuide 32개 중분류 섹터 대표주 선정(시총·이익률 Composite Rank).
> 채권 4종(국고채 3·10년, 회사채 AAA·BBB), 코인 10종(시총 층화추출).

## 회사명 마스킹 전략

실제 기업명이 뉴스·종토방에 노출되지 않도록 2단계 마스킹 적용:

1. **별칭 → 정식명** (매핑 테이블 + LLM fallback)
2. **정식명 → 가상명** (같은 받침 규칙 + 조사 자동 보정)

---

## 개발 규칙

- 데이터는 항상 Google Drive 영문 `Data/` 폴더 기준 (한글 `데이터/` 폴더 사용 금지)
- 데이터 미완성 시 stub JSON으로 프론트 개발 진행 (`seeds/import_news.js --stub`)
- 모든 설계 변경사항은 이 문서에 반영
- 스코프 기준은 **중간보고서**, 기술스택 기준은 본 문서

## 비고 — 프로토타입과의 차이

초기 프로토타입(이전 문서)은 4종목·5턴·초기자금 100만 원 수준이었으나, 본 문서는 **중간보고서 풀스코프(131자산·240턴·5,000만 원·스트레스/신뢰도/이벤트 포함)** 로 갱신한 것이다. 기술스택(React19+Vite·Express JS·PostgreSQL Docker·Python 파이프라인·2단계 마스킹)은 동일하다.
